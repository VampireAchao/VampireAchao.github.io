css继承

发表于2022-05-24|更新于2026-02-24
|浏览量:

浅水是喧哗的，深水是沉默的。——雪莱

我们知道CSS中一些属性会被继承：给父节点设置样式，子节点也会生效例如color、font-size之类的

但像widths, margins, padding, 和 borders 不会被继承，如果被继承，设想一下，给父节点加了一个border，里面的每个子孙元素都有一个border，这不是我们通常想要的效果

如何来控制这些属性呢？

CSS 为控制继承提供了四个特殊的通用属性值。每个css属性都接收这些值。

  • inherit

    设置该属性会使子元素属性和父元素相同。实际上，就是 “开启继承”.

  • initial

    设置属性值和浏览器默认样式相同。如果浏览器默认样式中未设置且该属性是自然继承的，那么会设置为 inherit 。

  • unset

    将属性重置为自然值，也就是如果属性是自然继承那么就是 inherit，否则和 initial一样
