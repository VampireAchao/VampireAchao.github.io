Zero date value prohibited

发表于2021-01-12|更新于2026-02-24
|浏览量:

世界上最大的谎言就是你不行。——《垫底辣妹》

报Zero date value prohibited错的时候

在连接mysql的URL上加入&zeroDateTimeBehavior=convertToNull即可

原因是因为我们存入数据库中的date类型字段有为0的数据

加了此参数，即可让mysql处理的时候，把为0的日期当作null处理
