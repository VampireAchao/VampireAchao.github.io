分享一个网站，可以加速`clone` `github`的仓库

https://www.gitclone.com/

例如：

```bash
git clone https://gitclone.com/github.com/AwakenCN/InChat.git
```

以及

```bash
git clone https://gitclone.com/github.com/ZEROKISEKI/hexo-theme-gal.git themes/gal
```

如果`clone`时出现`warning: You appear to have cloned an empty repository.`

删除掉本地`clone`下来的空文件，然后稍等下，多试试就可以了
