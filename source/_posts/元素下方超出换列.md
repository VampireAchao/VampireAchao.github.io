> 根本不应该为取悦别人而使自己失敬于人——卢俊

我们有时候需要这种需求：元素高度超出后换到下一列

则可以使用[column-count](https://developer.mozilla.org/zh-CN/docs/Web/CSS/column-count)

`column-count`可以指定最大列数

```css
column-count: 3;
column-count: auto;

column-count: inherit;
column-count: initial;
column-count: unset;
```

简写属性：[columns](https://developer.mozilla.org/zh-CN/docs/Web/CSS/columns)

```css
/* Column width */
columns: 18em;

/* Column count */
columns: auto;
columns: 2;

/* Both column width and count */
columns: 2 auto;
columns: auto 12em;
columns: auto auto;

/* Global values */
columns: inherit;
columns: initial;
columns: unset;
```

例子：

> ### [HTML](https://developer.mozilla.org/zh-CN/docs/Web/CSS/columns#html)
>
> ```html
> <p class="content-box">
>   This is a bunch of text split into three columns
>   using the CSS `columns` property. The text
>   is equally distributed over the columns.
> </p>
> ```
>
> ### [CSS](https://developer.mozilla.org/zh-CN/docs/Web/CSS/columns#css)
>
> ```css
> .content-box {
>   columns: 3 auto;
> }
> ```
>
> ### [Result](https://developer.mozilla.org/zh-CN/docs/Web/CSS/columns#result)
>
> <p class="content-box">
>   This is a bunch of text split into three columns
>   using the CSS `columns` property. The text
>   is equally distributed over the columns.
> </p>
>
> <style>
> .content-box {
>   columns: 3 auto;
> }    
> </style>
