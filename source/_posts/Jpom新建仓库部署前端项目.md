> 宽容并不等于放纵——俞吾金

安装`redis`

```shell
apt install redis
```

重新部署执行`sh`脚本

```shell
#!/bin/bash
BUILD_ID=DONTKILLME
function start(){
nohup java -jar '/test/management.jar' > '/test/management-log.txt' 2>&1 &
sleep 5s
exit
}
start
```

发现报错，由于`Ubunutu`使用`dash`代替了`bash`：

```shell
 Syntax error: "(" unexpected
```

按照[官方文档](https://jpom.top/pages/practice/project-dsl-java/#%E5%88%9B%E5%BB%BA%E8%84%9A%E6%9C%AC)执行

```shell
dpkg-reconfigure dash
```

然后在选择`YES`和`NO`的时候，选择`NO`即可

添加前端项目仓库：

![image-20221124143010263](/imgs/oss/picGo/image-20221124143010263.png)

构建列表添加构建步骤

```shell
cd ./management-web && yarn
cd ./management-web && yarn build:test
```

以及发布命令

```shell
cp -rp #{BUILD_RESULT_FILE} /test/manage-font/
```
