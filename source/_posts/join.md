`javscript`中 `join`用法如下：

![image-20210826194400276](/imgs/oss/picGo/image-20210826194400276.png)

```javascript
[1,2,3].join()		// 输出为1,2,3
```

不加参数的`join`默认使用`,`作为分隔符

我们可以指定分隔符为空字符串

```javascript
[1,2,3].join("")		// 输出为123
```

或者指定为别的字符等

```javascript
[1,2,3].join(".")		// 输出为1.2.3
```
