			let lazyLoadImgs = document.querySelectorAll("img")

			function lazyLoad() {
				// 屏幕可视区域高度
				let clientHeight = document.documentElement.clientHeight
				// 遍历图片
				lazyLoadImgs.forEach(e => {
					if (!e.src) {
						// 获取元素界面信息(宽高位置等)
						let rect = e.getBoundingClientRect()
						if (rect.top < clientHeight) {
							// 如果当前图片处于可视区域,则将data-src中的值放入src
							e.src = e.dataset.src
						}
					}
				})
				// 滚动时加载
				window.onscroll ??= lazyLoad
			}
			lazyLoad()
		</script>

在你们自己的代码中使用方式也很简单：

首先复制这串`js`：

```javascript

```

然后在你们页面中使用，这里需要将你们原来页面上`img`标签的`src`改为`data-src`，这样在元素处于可视区域,则将`data-src`中的值放入`src`，然后达到懒加载的效果
