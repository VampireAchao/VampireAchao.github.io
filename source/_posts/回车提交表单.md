> 却是平流无石处，时时闻说有沉沦。——唐•杜荀鹤

我们在页面开发中经常会写一些表单，但身为一个懒人，肯定是懒得点提交按钮的，我一般直接按回车提交

当我们的`form`满足只有一个`input`的时候，回车事件是生效的

```html
		<form>
			<input/>
		</form>
```

当我们的`form`有多个`input`的时候，回车事件失效了

```html
		<form>
			<input/>
			<input/>
		</form>
```

此时我们可以加一个`input`，让`type=submit`

即可再次满足回车提交效果

```html
		<form>
			<input/>
			<input/>
			<input type="submit"/>
		</form>
```

如果要隐藏掉，使用样式`display:hide`隐藏即可

但有时如果我们如果需要回车不提交表单，则可以避免上述条件

如果想要不执行`submit`事件，可以直接在`form`上加一个`onsubmit="return false"`即可

```html
		<form onsubmit="return false">
			<input/>
			<input/>
			<input type="submit"/>
		</form>
```
