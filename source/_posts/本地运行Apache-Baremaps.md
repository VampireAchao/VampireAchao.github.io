> 借了别人的钱，就是进了别人的网。——约翰·雷

`clone`代码

```bash
git clone https://github.com/apache/incubator-baremaps.git
```

运行：

```bash
GithubIireAchao:openstreetmap achao$ sdk use java 17.0.12-amzn

Using java version 17.0.12-amzn in this shell.
```

这里注意用的`sdkman`[sdkman | 阿超](https://vampireachao.github.io/2024/05/20/sdkman/)

然后构建一下

```bash
GithubIireAchao:openstreetmap achao$ pwd
/Users/achao/IdeaProjects/incubator-baremaps
GithubIireAchao:openstreetmap achao$ ./mvnw clean install
```

然后是`docker`运行一下，拉不下来镜像用这个：

[docker-desktop配置腾讯云镜像 | 阿超](https://vampireachao.github.io/2024/08/05/docker-desktop%E9%85%8D%E7%BD%AE%E8%85%BE%E8%AE%AF%E4%BA%91%E9%95%9C%E5%83%8F/)

```bash
GithubIireAchao:incubator-baremaps achao$ docker run   --name baremaps   --publish 5432:5432   -e POSTGRES_DB=baremaps   -e POSTGRES_USER=baremaps   -e POSTGRES_PASSWORD=baremaps   -d postgis/postgis:latest
```

然后解压：

```bash
GithubIireAchao:incubator-baremaps achao$ tar -xvf ./baremaps-cli/target/apache-baremaps-0.7.4-SNAPSHOT-incubating-bin.tar.gz -C target/
```

然后配置环境变量，路径记得对一下

```bash
GithubIireAchao:incubator-baremaps achao$ export PATH=$PATH:/Users/achao/IdeaProjects/incubator-baremaps/target/apache-baremaps-0.7.4-SNAPSHOT-incubating-bin/bin/
GithubIireAchao:openstreetmap achao$ baremaps -V
Baremaps v0.7.4-SNAPSHOT
```

然后进入目录

```bash
GithubIireAchao:incubator-baremaps achao$ cd examples/openstreetmap
```

刷一下数据

```bash
GithubIireAchao:openstreetmap achao$ baremaps workflow execute --file workflow.json
GithubIireAchao:openstreetmap achao$ baremaps map dev   --tileset 'tileset.json'   --style 'style.json'
```
