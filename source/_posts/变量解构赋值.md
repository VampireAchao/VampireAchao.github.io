它这里用到了一个`ES6`的新特性：解构赋值

这里简单复现一下

```javascript
			var param = {
				columns: {
					property: "我是property"
				},
				data: ["我是", "data", "数组"]
			}
			var {columns,data} = param
			console.log(columns)
			console.log(data)
```

输出结果

![image-20210503103425445](/imgs/oss/picGo/image-20210503103425445.png)

可以看到我们这里使用

```javascript
			var {columns,data} = param
```

去拿到了`param`里的变量`columns`和`data`

这里注意一点，如果我们使用的名字不同的话，是无法取得里面的属性的

例如

```javascript
			var param = {
				columns: {
					property: "我是property"
				},
				data: ["我是", "data", "数组"]
			}
			var {col,data} = param
			console.log(col)
			console.log(data)
```

可以看到输出`undefined`了

![image-20210503103716726](/imgs/oss/picGo/image-20210503103716726.png)

那如果我们这个变量在上面有同名的怎么办呢，我们可以使用冒号取个别名

```javascript
			var columns = "我被定义了"
			var param = {
				columns: {
					property: "我是property"
				},
				data: ["我是", "data", "数组"]
			}
			var {columns:col, data} = param
			console.log(columns)
			console.log(col)
			console.log(data)
```
