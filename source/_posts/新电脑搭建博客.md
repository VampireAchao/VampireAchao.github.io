![image-20200920131654806](/imgs/oss/picGo/image-20200920131654806.png)

![image-20200920132012107](/imgs/oss/picGo/image-20200920132012107.png)

下载`node12`（14之后会报警告，虽然可以用，但警告看上去很烦。。。）

![image-20200920142516264](/imgs/oss/picGo/image-20200920142516264.png)

```shell
#查看node版本

#查看npm版本
npm -v
```

![image-20200920142618893](/imgs/oss/picGo/image-20200920142618893.png)


![image-20200920142834342](/imgs/oss/picGo/image-20200920142834342.png)

[主题挑选网址](https://hexo.io/themes/)

比如我现在用的这个

![image-20200920154044816](/imgs/oss/picGo/image-20200920154044816.png)

![image-20200920154301873](/imgs/oss/picGo/image-20200920154301873.png)

```shell

```

![image-20200920154504279](/imgs/oss/picGo/image-20200920154504279.png)

下载完成后

```shell
cnpm install hexo-renderer-sass --save
```

![image-20200920161009391](/imgs/oss/picGo/image-20200920161009391.png)

然后

```shell
cnpm install hexo-renderer-scss --save
```

![image-20200920161045229](/imgs/oss/picGo/image-20200920161045229.png)

```shell
cnpm install hexo-generator-json-content --save
```

![image-20200920161142766](/imgs/oss/picGo/image-20200920161142766.png)

然后进入`_config.yml`

粘贴

```yml
jsonContent:
  dateFormat: MM-DD
  pages:
    title: true
    text: true
    path: true
    date: true
    excerpt: true
    preview: true
  posts:
    title: true
    text: true
    path: true
    date: true
    excerpt: true
    tags: [{
      name: tag.name,
      slug: tag.slug,
      permalink: tag.permalink
    }]
    preview: true
```

并且修改主题为`gal`，保存
