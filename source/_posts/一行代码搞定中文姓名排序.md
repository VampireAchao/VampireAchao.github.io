上码！

两个参数分别为：一个字符串数组、一个接收比较器，以确定所述阵列的顺序

Collator是一个抽象基类，子类实现特定的整理策略。

```java
        Arrays.sort(names, Collator.getInstance(java.util.Locale.CHINA));
```

完整代码如下：

```java
String[] names = {"张三", "李四", "王五", "赵六", "狂神", "阿超"};
        Arrays.sort(names, Collator.getInstance(java.util.Locale.CHINA));
        for (String name : names) {
            System.out.println(name);
        }
```

输出结果：

```java
阿超
狂神
李四
王五
张三
赵六
```
