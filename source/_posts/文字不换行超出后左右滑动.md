如果我们想让它不换行，并且超出后能左右滑动

可以在`css`中加上如下代码

```css
white-space: nowrap;
overflow-x: scroll;
```

效果如下

![image-20210418141038030](/imgs/oss/picGo/image-20210418141038030.png)

如果我们还需隐藏滚动条

则再加上

```css
			::-webkit-scrollbar {
				width: 0;
				height: 0;
				color: transparent;
			}
```

即可

![image-20210418141135289](/imgs/oss/picGo/image-20210418141135289.png)
