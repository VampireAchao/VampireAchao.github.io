拓展函数的定义：

```kotlin
fun String.delBlank() = this.filter { !Character.isWhitespace(it) }
```

以及：

```kotlin
fun Int?.default(i: Int = 0) = i

fun String?.default(i: String = "") = i
```

使用：

```kotlin
fun String.delBlank() = this.filter { !Character.isWhitespace(it) }

val noBlank = "Hello World".delBlank()

println(noBlank)

fun Int?.default(i: Int = 0) = i

fun String?.default(i: String = "") = i

val s: Int? = null
println(s.default())
```
