朋友找我要装好`docker`的虚拟机，于是就装了一个啦

```shell
yum install docker
```

![image-20210303215515790](/imgs/oss/picGo/image-20210303215515790.png)

提示`Y/N`我们一直输入`y`然后回车就行了

最后看到`Complete`即可

![image-20210303215704492](/imgs/oss/picGo/image-20210303215704492.png)

我们可以输入

```shell
docker -v
```

查看版本
