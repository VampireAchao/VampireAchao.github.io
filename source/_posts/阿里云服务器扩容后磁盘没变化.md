> “品德”的自然位置就在“自由”的近旁，但是离开“极端自由”，和“奴役”却都是同样的遥远——孟德斯鸠

今天在`ubuntu`扩容完毕后执行`df -h`发现根本没变化

![image-20230223210521590](/imgs/oss/blog/vampireachao/image-20230223210521590.png)

查看文档告诉还需要调整分区和文件系统

执行：

```shell
resize2fs /dev/vda3
```

结果提示` 提示The filesystem is already 10434043 (4k) blocks long. Nothing to do!`

问了客服，客服说需要执行

```shell
apt-get update
type growpart || apt-get install -y cloud-guest-utils
LC_ALL=en_US.UTF-8 growpart /dev/vda3
resize2fs /dev/vda3
```

挨个执行了过去，然后再执行`df -h`就好了
