title: 根据进程名模糊查询并结束
date: 2022-04-19 12:55:29

> 酒杯里竟能蹦出友谊来。——盖伊

使用方式非常简单

```shell
ps -ef | grep [进程名] | awk '{print $2}' | xargs kill -9
```

例如

```shell
ps -ef | grep java | awk '{print $2}' | xargs kill -9
```

注意如果在`jenkins`

我们需要使用`|| true`忽略错误，避免因为没找到对应进程构建失败

```shell
ps -ef | grep java | awk '{print $2}' | xargs kill -9 || true
```

顺带再聊一个查找目录的命令

```shell
find [路径] -name [名字]
```

例如

```shell
find / -name java
```
