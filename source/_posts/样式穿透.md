最后发现都是徒劳，样式根本没有加上

所以这里我们安装`less`

注意一定要在项目没运行的时候安装

```shell
cnpm i less --save-dev
cnpm i -s less-loader
cnpm i -g less
```

然后在`vue.config.js`中加上配置

```javascript
module.exports = {
    devServer: {
        port: 3000
    },
    css: {
        loaderOptions: {
            less: {
                modifyVars: {},
                javascriptEnabled: true,
            }
        }
    },
}
```

最后在`style`标签加上`lang="less"`以及在类选择器前方加个`/deep/`

```vue
<style scoped lang="less">
/deep/ .cell {
  background-color: #9a9a9a;
  color: #ffffff;
}
</style>
```
