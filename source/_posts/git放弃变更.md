> 度之住事,验之来事,参之平素，可则决之。一一《鬼谷子》

我们有时想放弃本地的变更，直接从远端仓库`pull`代码到本地覆盖

可以使用

```shell
git checkout -f
```

去放弃所有变更

当然，`-f`可以使用文件(夹)路径替换，这样的话只放弃指定的文件(夹)中的变更

但如果你新增了文件，想要放弃新增则可以使用

```shell
git clean -df
```

如果需要指定文件(夹)，则可以在其中加入文件(夹)名，例如

```shell
git clean xxx.txt -df
```

然后就可以`pull`覆盖本地啦
