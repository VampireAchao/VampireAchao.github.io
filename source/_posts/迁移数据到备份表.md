然后停止了下来，换另一种方式，将不需要删除的数据迁移到备份表，然后删除原表，再重命名回去

```sql
CREATE TABLE mx_product_detail_bak LIKE mx_product_detail;
INSERT INTO mx_product_detail_bak
SELECT *
FROM mx_product_detail
WHERE type = 0;
DROP TABLE mx_product_detail;
RENAME TABLE mx_product_detail_bak TO mx_product_detail;
```
