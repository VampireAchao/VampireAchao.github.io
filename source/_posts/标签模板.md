按照[阮一峰`ES6`入门](https://es6.ruanyifeng.com/#docs/string#%E6%A0%87%E7%AD%BE%E6%A8%A1%E6%9D%BF)中写到的

我们可以使用函数名+模板字符串写法来调用函数，例如：

定义这样一个函数

```javascript
function templateFun(arguments){
    return `Hello ${arguments[0]}!`
}
```

这里简单用模板字符串，返回`Hello `+参数数组第一项元素+`!`

我们普通调用写法：

```javascript
templateFun(['Ruben'])
```

输出`'Hello Ruben!'`

我们如果使用标签模板，写法如下：

```javascript
templateFun`Ruben`
```

![image-20220205165430021](/imgs/oss/picGo/image-20220205165430021.png)

如果其中有变量，则会进行剥离拆分，我们对函数稍加修改，参数改为可变参数

```javascript
function templateFun(...arguments){
    console.log(arguments)
    return `Hello ${arguments}!`
}
templateFun`Ruben${'Happy'}快乐${'new'}新${'year'}年`
```

调用结果为：

![image-20220205171530286](/imgs/oss/picGo/image-20220205171530286.png)

可以看到变量将我们其中的字符串隔开，组成头部的数组，而`raw`中则是我们的变量
