`springboot`默认使用`jackson`进行序列化

如果我们想使用`fastJson`

则可以注入一个`HttpMessageConverters`

```java
    @Bean
    public HttpMessageConverters fastJsonHttpMessageConverters(){
        return new HttpMessageConverters(new FastJsonHttpMessageConverter());
    }
```

我们测试一下

给我们的`bean`中属性`keywords`加上注解`@JSONField(name = "keyword")`
