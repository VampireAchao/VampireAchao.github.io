4. 如果是外部 TypeScript 库

5. 常见问题

  • 没有类型提示/报错？
    请检查 tsconfig.json 是否包含了 TypeScript 文件目录。
  • JS 文件中 import TS 文件出错？
    allowJs: true 可以支持 js/ts 混用。
  • Vetur/Volar 报错？
    可以考虑升级相关插件或者切换到 Volar（推荐新项目用 Volar）。
