高性能：专注于在所有相关浏览器中实现出色的性能，具有最少的依赖项

可及性：旨在符合 WCAG 2.2 AA 标准，以创建每个人都可以使用的组件

灵活性：通过 Web Components 和 JavaScript 类提供解决方案，这些类可以使用、采用和扩展以满足所有需求

现代代码：Lion 以纯 es 模块的形式分发
