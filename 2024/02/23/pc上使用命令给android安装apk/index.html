adb install е‘Ҫд»Өз”ЁдәҺйҖҡиҝҮ Android Debug Bridge (ADB) е°Ҷеә”з”ЁзЁӢеәҸпјҲйҖҡеёёжҳҜ APK ж–Үд»¶пјүе®үиЈ…еҲ°иҝһжҺҘзҡ„ Android и®ҫеӨҮжҲ–жЁЎжӢҹеҷЁдёҠгҖӮиҝҷжҳҜ Android еә”з”ЁејҖеҸ‘е’ҢжөӢиҜ•иҝҮзЁӢдёӯеёёз”Ёзҡ„е‘Ҫд»ӨпјҢе…Ғи®ёејҖеҸ‘иҖ…зӣҙжҺҘд»ҺејҖеҸ‘зҺҜеўғе°Ҷеә”з”Ёе®үиЈ…еҲ°и®ҫеӨҮдёҠгҖӮ

еҸӮж•°пјҲOptionsпјү

adb install е‘Ҫд»Өж”ҜжҢҒеӨҡдёӘйҖүйЎ№пјҢеҸҜд»Ҙдҝ®ж”№е®үиЈ…иЎҢдёәпјҡ

  • -l: е°Ҷеә”з”Ёе®үиЈ…еҲ°дҝқжҠӨзӣ®еҪ•дёӢгҖӮ
  • -r: йҮҚж–°е®үиЈ…зҺ°жңүеә”з”ЁпјҢдҝқз•ҷе…¶ж•°жҚ®гҖӮ
  • -t: е…Ғи®ёе®үиЈ…жөӢиҜ• APKгҖӮ
  • -s: е°Ҷеә”з”Ёе®үиЈ…еҲ° SD еҚЎгҖӮ
  • -d: е…Ғи®ёйҷҚзә§иҰҶзӣ–е®үиЈ…гҖӮ
  • -g: жҺҲдәҲжүҖжңүиҝҗиЎҢж—¶жқғйҷҗгҖӮ
  • --abi <ABI>: ејәеҲ¶дҪҝз”ЁжҢҮе®ҡзҡ„ ABI иҝӣиЎҢе®үиЈ…гҖӮиҝҷеңЁдҪ зҡ„ APK ж”ҜжҢҒеӨҡз§Қ ABIпјҢдҪҶи®ҫеӨҮж”ҜжҢҒзҡ„ ABI дёҚжҳҜ APK зҡ„й»ҳи®Ө ABI ж—¶еҫҲжңүз”ЁгҖӮ
  • --instant: е°Ҷеә”з”ЁдҪңдёә Instant App е®үиЈ…гҖӮ
  • --no-streaming: йҖҡиҝҮ USB е®үиЈ…еӨ§ APK ж—¶дёҚдҪҝз”ЁжөҒејҸдј иҫ“гҖӮ
  • --fastdeploy: дҪҝз”Ё Fast Deploy жӣҙж–°еә”з”ЁгҖӮ
  • --incremental: дҪҝз”ЁеўһйҮҸжӣҙж–°е®үиЈ… APKгҖӮ
  • --force-agent: дҪҝз”Ё Fast Deploy ејәеҲ¶дҪҝз”Ёе®үиЈ…д»ЈзҗҶпјҢеҚідҪҝ APK жІЎжңүж”№еҸҳгҖӮ
  • --no-restart: е®үиЈ… APK еҗҺдёҚйҮҚеҗҜеә”з”ЁгҖӮ
  • --no-cache: е®үиЈ…еә”з”Ёж—¶дёҚдҪҝз”Ёе®үиЈ…еҷЁзј“еӯҳгҖӮ
  • --force-queryable: е®үиЈ…дёҚеҸҜжҹҘиҜўзҡ„еә”з”ЁгҖӮ
  • --wait: е®үиЈ…е®ҢжҲҗеҗҺзӯүеҫ…пјҢзӣҙеҲ°и®ҫеӨҮеҮҶеӨҮе°ұз»ӘгҖӮ
