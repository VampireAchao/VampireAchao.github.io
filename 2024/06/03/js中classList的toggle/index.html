在 HTML 中实现白天和夜晚主题切换

  3. JavaScript 功能：

    • <button> 按钮用于触发主题切换。

  2. CSS 样式：

    • .night 类定义了夜晚模式的样式，背景颜色为深灰色，文本颜色为浅灰色。

    • toggleTheme 函数通过 classList.toggle 方法切换 body 元素的 night 类，从而实现主题切换。
