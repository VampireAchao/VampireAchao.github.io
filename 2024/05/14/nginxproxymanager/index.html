该项目作为预构建的 docker 映像提供，使您能够轻松转发到在家或其他地方运行的网站，包括免费的 SSL，而无需了解太多有关 Nginx 或 Letsencrypt 的信息。

  • 使用 Let’s Encrypt 免费 SSL 或提供您自己的自定义 SSL 证书

  • 主机的访问列表和基本 HTTP 身份验证
