  3. Functional Programming features：指函数式编程的特性，如高阶函数、纯函数、不可变性等。

  6. Exception Handling：指程序出现异常时的处理机制，可以用于优雅地处理错误。

  7. Advanced Assignment：指高级赋值语法，如解构赋值、链式赋值等。
