重新执行发现拉取不到org.apache.streampark:streampark-shaded-slf4j

在streampark-shaded下面的pom.xml右键

选择Add as Maven Project

这里dist其实可以去掉

然后再回到项目目录执行

解决办法：首先右键pom.xml选择Unlink Maven Projects

然后删除项目下的.idea并重新打开项目

重新执行编译命令即可正常启动
